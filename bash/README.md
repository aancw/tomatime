# tomatime bash script

## Dependencies
  - bash / sh
  - libnotify (notify-send)

## Penggunaan:
  - jalankan tomatime
  - input jumlah pomodoro
  - selamat bekerja

## Konfigurasi
Buka file `tomatime` dan edit baris 8 - 10
``` bash
kerja=25;
istirahat_pendek=5;
istirahat_panjang=20;
```
angka yang diinputkan adalah jumlah menit yang akan digunakan.
